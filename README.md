# Monster Hunter Wilds Build Calculator

A Python-based build calculator for Monster Hunter Wilds.

## Status
Working v1.0 uploaded. The project is currently around 50% complete and actively under development. A more modular and scalable version is also being built.

## Features
- Weapon selection by type and rarity
- Armor selection by slot and rarity
- Decoration filtering based on slot level and allowed gear
- JSON-based data system for weapons, armor, and skills

## Tech Stack
- Python
- PySide6 (GUI)
- JSON (data storage)

## Design Notes
This version intentionally uses a minimal, unstyled interface.  
No advanced styling or color system has been implemented yet, as the current focus is on core functionality and system structure.

UI styling, visual feedback, and color systems are planned for future updates.

## In Progress
- Build calculation logic (damage, stats, optimization)
- UI improvements and visual styling
- Refactoring into a more modular architecture
